Установка агента

Заббикс агент используем на всех машинах с Windows.

1)Необходимо скачать на устройство агента с официального сайта.


После перехода на сайт, спускаемся в самый низ и находим следующее:

Нам нужна самая модная, последняя версия (в данном случае Zabbix agent 2).

Нажимаем Download.

После запуска установщика нажимаем Далее (Next), соглашаемся с условиями пользования и жмем еще раз Далее

Здесь ничего не меняем, жмем Next

В следующем окне важным пунктом является Zabbix Server IP/DNS

Вводим IP-адрес на котором работает ZABBIX У нас он 10.0.1.29 и жмем Next -> Install

Можем проверить что агент работает, зайдя в диспетчер задач -> Службы, обычно в самом конце списка появится Zabbix agent


Добавление устойства в Zabbix


2)Перейдем к Заббиксу

Для этого в адресной строке любого браузера введем ip-адрес zabbix сервера (10.0.1.29)

Нажимаем Сбор данных -> Узлы сети и сверху кнопка «Создать узел сети»

В шаблонах необходимо указать «Windows by Zabbix agent», в группах узлов сети - группу к которой вы хотите его добавить, в моем случае компьютер будет в группе HOSTS (Имя может быть любым)


Самое важное, это указать IP-адрес компьютера, который будем мониторить. Он должен быть неизменным, у меня это 10.0.1.100 В графе порт оставляем 10500, без имзенений. Жмем добавить и ВСЕ!
